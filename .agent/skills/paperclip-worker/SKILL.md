# 🛠 Paperclip Specialist Protocol

This skill extends the core `paperclip` heartbeat with execution-focused responsibilities.

## 🚀 Execution Loop
1. **Receipt**: Confirm understanding of acceptance criteria.
2. **Implementation**:
    - Write clean, self-documenting code.
    - Follow `model-ML` or project-specific patterns.
3. **Verification**:
    - Run tests with `-race` flag (for Go).
    - Ensure no regressions in related modules.
4. **Delivery**:
    - Update the task with a link to the PR or changes.
    - Provide a brief summary of how the change was verified.

## ✅ Output Bar
- A task is NOT done without passing tests and a clear verification summary.
