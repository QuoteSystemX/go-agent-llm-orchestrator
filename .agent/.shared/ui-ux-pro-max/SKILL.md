## Overview

The `ui-ux-pro-max` skill provides advanced design capabilities for creating premium web interfaces. It enforces design tokens, accessibility standards, and modern aesthetics.

## Scripts

- `core.py`: Main engine for generating design tokens and coordinating component layouts.
- `search.py`: Specialized design discovery tool for finding harmonious color palettes and typography pairings.
- `design_system.py`: Manages the application of the chosen style to the component library.

## Usage

Use this skill when you need to "WOW" the user with a premium design. 
Trigger with `/ui-ux-pro-max` or by referencing the skill in your task.

## Rules

1. **Purple Ban**: Do not use purple or violet shades unless explicitly requested.
2. **Template Ban**: Avoid standard "header-sidebar-footer" layouts. Use unique, dynamic compositions.
3. **Typography**: Always use Google Fonts (Inter, Roboto, Outfit) instead of browser defaults.
